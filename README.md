# 电商项目（Spring Boot）

> 课程项目 | 2025.12 - 2026.02 | 独立完成（AI辅助编码）

## 项目简介

基于 Spring Boot 的 B2C 电商平台，实现**用户端、商家端、管理端**三端分离，覆盖商品浏览、购物车、订单交易、数据统计等核心电商业务。

## 技术栈

| 分类 | 技术 |
|------|------|
| 后端框架 | Spring Boot |
| 持久层 | MyBatis |
| 数据库 | MySQL |
| 前端 | HTML5、CSS3、JavaScript、ECharts |
| 构建工具 | Maven、Git |

## 核心功能

| 角色 | 主要功能 |
|------|----------|
| 普通用户 | 商品浏览、购物车、下单、支付、确认收货、评价、售后 |
| 商家 | 商品管理（增删改查、上下架）、订单处理、发货、销售数据统计 |
| 管理员 | 用户管理、商家审核、投诉处理、违规管理 |

## 项目亮点

- **数据隔离**：通过 `merchant_id` 字段实现商家数据隔离
- **订单状态机**：设计完整订单状态流转
- **自动确认收货**：定时任务自动完成
- **测试验证**：设计并执行 30+ 个测试场景

## 本地运行

1. 创建 MySQL 数据库：`CREATE DATABASE shop`
2. 导入 SQL 文件
3. 修改数据库密码
4. 运行项目
5. 访问 `http://localhost:8088`
